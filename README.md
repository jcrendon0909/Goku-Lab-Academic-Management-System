
  # Goku Lab Academic Management System

Sistema desarrollado para la academia Algorithmics Goku Lab, diseñada para optimizar la administración académica y financiera mediante un sistema centralizado y seguro.

La plataforma cuenta con autenticación de usuarios (login) y herramientas para la gestión de clases, permitiendo agendar y reagendar sesiones de manera eficiente. Además, incorpora un calendario interactivo para el seguimiento de actividades académicas y módulos administrativos para el control de pagos pendientes y consulta del historial de pagos de los estudiantes.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
